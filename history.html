History of HTML

The evolution of the web's foundational language

1989

Tim Berners-Lee invents the World Wide Web while at CERN. HTML is proposed as a means of creating and linking documents.

1991

The first public description of HTML is made available by Tim Berners-Lee, defining 18 elements.

1995

HTML 2.0 is published as an IETF specification, providing a standard that expands on the original 18 elements.

1997

HTML 3.2 is released by the W3C, introducing new elements and attributes to support a wider range of web pages.

1999

HTML 4.01 becomes a W3C recommendation, offering enhanced support for scripting, stylesheets, and accessibility.

2008

HTML5 is introduced, aiming to improve the language with support for the latest multimedia while keeping it easily readable by humans and consistently understood by computers and devices.

2014

HTML5 becomes a W3C recommendation, solidifying its use as the standard for modern web development.
